# Data-Science-job-salaries 
📌 Project Title / Headline
Data Science Job Salary Analytics Dashboard
An Interactive Power BI Dashboard for Analyzing Global Data Science Salaries & Workforce Trends (2020–2022)
📖 Short Description / Purpose
The Data Science Job Salary Analytics Dashboard is a Power BI project designed to analyze salary trends and workforce distribution in the Data Science industry.
This dashboard provides insights into how salaries vary based on:
Experience level
Job title
Employment type
Company size
Remote work category
Geographic location
The project helps users understand compensation trends in the global Data Science job market through interactive and visually appealing reports.
🛠️ Tech Stack
The following tools and technologies were used to build this project:
Power BI → Dashboard creation & data visualization
Microsoft Excel / CSV → Data source and preprocessing
Power Query → Data cleaning and transformation
DAX (Data Analysis Expressions) → KPI calculations and measures
Data Visualization Techniques → Charts, maps, KPIs, treemaps, and trend analysis
